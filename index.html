Projects

I came up with the idea for a web app to help English teachers to teach speaking to students in the classroom by simultaneously recording student conversations with teacher feedback given over the keyboard.

The app uses Action Cable to create live speaking sessions which both the teacher and student can simultaneously join. The student records the session on their smartphone while the teacher records feedback on speaking mistakes. The audio file is uploaded to a cloud-based storage service and a record of this and the student's mistakes are saved in the database.

We built an Airbnb-style marketplace app that allows people to book haircut appointments with barbers in Tokyo.

To populate our database, we used Yelp's API Business Endpoint to get shop information for over 50 barbers in Tokyo. The app included two user models, customers and barbers, which were both implemented with the Rails gem Devise.
